# tetris
simple tetris game finished with XiaoY's help

##get start

*.class now exists already and because I use eclipse to finish this game, so I recommend you use that to find Main.java to run or load all source code.

for db aspect I use mysql to connect with jdbc, so you could use this function with your local db, either modify data/cfg.xml param which in __data__ tag.

##directly play

on dir __export__ you could directly exec __Tetris.jar__ or start.bat to play game.

more info please refer user guide in __export/guide.txt__
